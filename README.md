# 一林的求职主页

## 怎么改（超简单）

1. 打开文件：`contents/改这里.yml`
2. 按里面的中文注释改文字
3. 保存后刷新网页

**只需要改这一个文件。**

### 常见修改

- 改名字 / 职位 / 一句话介绍 → 文件最上面
- 加教育 / 实习 / 项目 → 复制对应一整块；删掉一整块就减少；写几条页面就显示几条
- 不想展示教育或实习 → 把对应的 `显示` 改成 `否`
- 改技能 → 改「技能」里的分类和列表
- 改邮箱 → 改「邮箱」那一行
- 改背景图 → 把图片放进 `static/assets/background/`，再改「背景图」列表

没有演示链接时，写成：`演示链接: 无`

### 背景图尺寸

- **推荐：1920 × 600 像素**（宽 × 高）
- 也可以：1920×500 或 2400×700
- 要用横向长图；重要内容放中间，上下左右可能被裁切
- 格式：jpg / png / webp，单张建议 < 800KB

### 头像尺寸

替换 `static/assets/img/photo.jpeg`  
推荐：**400 × 400** 正方形

## 本地预览

```bash
npx serve .
```
